== Description ==

Easy way to add after content widgets. Supports posts, pages and custom post types. Light and flexible.

== Installation ==

1. Download and activate the plugin through the **Plugins** menu in WordPress dashboard.
2. Place your widgets in the widget areas of your choice.
3. Visit any singular post, page or custom post type post and enable the display of content widgets for this post type in **Customizer** -> **Content widgets**.
4. You are ready to go!
